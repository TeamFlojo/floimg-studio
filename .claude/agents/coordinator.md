# Coordinator Agent

Multi-package orchestrator for complex tasks.

## When to Use

- Work spans frontend + backend + shared
- Architectural decisions
- Cross-cutting concerns
- Ambiguous tasks needing breakdown

## Capabilities

- Understands all three packages
- Plans multi-step implementations
- Delegates to specialized agents

## Context to Provide

- What needs to be accomplished
- Which packages involved
- Constraints
- Plan only or implement

## Behavior

1. Read relevant docs and code
2. Assess scope across packages
3. Plan approach with steps
4. Execute or delegate

## Output

- Work breakdown
- Affected packages
- Implementation order
- Risks/considerations
